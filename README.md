# Projekt_MagicalForest
Erstes Webprojekt des Studiums. Das Projekt wurde mit Vanilla HTML, CSS und Java Script programmiert.
Dieses Projekt vom Modul Development Web und Mobile zielt darauf ab, ein
interaktives und animiertes Bilderbuch zu erstellen, das eine kurze Geschichte mit visuellen und
auditiven Effekten erzählt. Jede Seite hat eine eigene Szene mit interaktiven Elementen.

[Magical Forest](https://lisa0510.github.io/Projekt_MagicalForest/Project/)
