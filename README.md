# Projekt_MagicalForest
Erstes Webprojekt des Studiums. Das Projekt wurde mit Vanilla HTML, CSS und Java Script programmiert.
Dieses Projekt vom Modul Development Web und Mobile zielt darauf ab, ein
interaktives und animiertes Bilderbuch zu erstellen, das eine kurze Geschichte mit visuellen und
auditiven Effekten erzählt. Jede Seite hat eine eigene Szene mit interaktiven Elementen.

[Magical Forest](https://lisa0510.github.io/Projekt_MagicalForest/Project/)

Credits an meine lieben Voice Actors für die Charaktere.

Für mein DeWeb-Projekt habe ich mich entschieden, für jede Szene eine individuelle Hintergrundmusik zu wählen, um die Atmosphäre und Stimmung jeder Szene zu verstärken. Die Musikstücke habe ich sorgfältig von Pixabay ausgewählt und heruntergeladen, um sicherzustellen, dass sie rechtlich und kostenlos zur Verfügung stehen. Zusätzlich zu den Hintergrundmusiken habe ich spezielle Sounddesigns für mein Projekt erstellt, um den einzelnen NPC-Charakteren mehr Tiefe und Persönlichkeit zu verleihen. Diese Sounds wurden von mir selbst aufgenommen und bearbeitet, um sicherzustellen, dass sie perfekt zu den Charakteren und Szenen passen.
